# 📊 Superstore Data Analytics

## 📌 Overview
This project analyzes the **Superstore dataset** to uncover insights on sales, profit, and customer trends.  
It demonstrates skills in **SQL querying, data cleaning, visualization, and dashboard design** using Power BI.

---

## 📂 Repository Structure
- **data/** → Raw and cleaned datasets  
- **sql/** → SQL scripts and query outputs  
- **dashboards/** → Power BI dashboards and visuals  
- **reports/** → Business insights and documentation  
- **README.md** → Project overview  

---

## 🔑 Key Insights
- **Top-performing categories:** Technology drives the highest revenue.  
- **Regional trends:** The West region shows the strongest profitability.  
- **Customer segments:** Corporate customers contribute consistent profits.  
- **Seasonality:** Q4 sales spikes suggest opportunities for targeted promotions.  

---

## 🛠 Tools & Technologies
- **SQL** → Data cleaning, aggregation, and querying  
- **Power BI** → Interactive dashboards and visual storytelling  
- **Excel** → Data preprocessing and validation  

---

## 🚀 Highlights
- Interactive dashboards for sales & profit analysis  
- SQL queries for data cleaning and aggregation  
- Business insights report summarizing findings  
- Professional repo structure for easy navigation
